# Server

```bash title="Terminal" 
pip3 install "fastapi[standard]"
```

### pnpm dev

```bash
python3 -m fastapi dev main.py
```

### package.json

```bash
pip3 freeze > requirements.txt
```

### Prettier

```bash
pip3 install black
python3 -m black main.py
```

## Types

```python title="main.py"
from typing import Annotated, Set, Union, List, Dict
from typing_extensions import Annotated
from uuid import UUID
from datetime import datetime, time, timedelta

from fastapi import Query, Path, Body
from pydantic import BaseModel, HttpUrl, Field
```

> [Search](https://fastapi.tiangolo.com/python-types/) them if you don't know


> **Good to know**
> - If you want to add new things to the returned data by using the body data, make sure to use `.model_dump()`, otherwise a thing can't read it.
